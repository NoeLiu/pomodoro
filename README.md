# 番茄时钟 (Pomodoro Timer)

一个简洁的Windows番茄时钟桌面应用。

## 功能特点

- 25分钟专注工作时间
- 5分钟短休息
- 15分钟长休息（每4个番茄钟后）
- 开始/暂停/重置/跳过控制
- 番茄钟计数
- 计时器显示

## 编译运行

```bash
gcc pomodoro.c -o pomodoro.exe -mwindows
./pomodoro.exe
```

需要安装 MinGW-w64 编译器。

## 操作说明

1. 点击"开始"按钮开始计时
2. 工作时间结束后显示消息框
3. 自动切换到休息时间
4. 使用"重置"按钮重新开始当前周期
5. 使用"跳过"按钮跳到下一阶段
